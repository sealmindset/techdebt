# Changelog

## [0.1.0] - 2026-03-31

### Added
- Initial project setup with FastAPI + Next.js scaffold
- OIDC authentication with mock-oidc for local development
- Database-driven RBAC with 5 roles (Super Admin, Admin, Finance Reviewer, Department Head, Viewer)
- Application portfolio management with ~100 seeded apps across categories
- Dashboard with key metrics (total apps, spend, adoption rates, AI recommendations)
- App detail pages with usage stats, cost per user, adoption rates
- AI-powered rationalization recommendations (keep/cut/replace/consolidate)
- Decision workflow with structured justification and approval chain
- Data source integrations (Workday, AuditBoard, Entra ID) with mock services
- Activity logging with admin UI
- Application settings management
- Standard UI components (DataTable, Breadcrumbs, QuickSearch, ModeToggle)
