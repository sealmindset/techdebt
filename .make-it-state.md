## What Was Built
- FastAPI + Next.js web application for SaaS/software rationalization
- OIDC authentication with mock-oidc (5 test users)
- Database-driven RBAC with 5 roles and 31 permissions
- 100 seeded applications across 10 categories
- 60 AI recommendations (keep/cut/replace/consolidate)
- 15 decisions with approval workflow
- Dashboard with metrics (total apps, spend, adoption, savings)
- 7 domain pages (Dashboard, Applications, App Detail, Recommendations, Decisions, Data Sources, Submissions)
- 4 admin pages (Users, Roles, Settings, Activity Logs)
- 3 mock data source services (Workday, AuditBoard, Entra ID)
- Docker Compose with 7 services (frontend, backend, db, mock-oidc, mock-workday, mock-auditboard, mock-entra)
